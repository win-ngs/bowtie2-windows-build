Third-Party Notices for Bowtie 2 2.5.5 Windows UCRT64 Build
===========================================================

This file summarizes third-party components included in, or used to build, the
Windows release ZIP for this repository.

This is an informational notice file, not legal advice. Each third-party
component remains under its own license terms.


Packaged runtime files
----------------------

The release ZIP includes Bowtie 2 executables and Windows wrapper scripts:

  bowtie2-align-s.exe
  bowtie2-align-l.exe
  bowtie2-build-s.exe
  bowtie2-build-l.exe
  bowtie2-inspect-s.exe
  bowtie2-inspect-l.exe
  bowtie2.cmd
  bowtie2-build.cmd
  bowtie2-inspect.cmd
  scripts/bowtie2.ps1
  scripts/bowtie2-build.ps1
  scripts/bowtie2-inspect.ps1

The tested UCRT64 build was linked with:

  -static-libgcc -static-libstdc++ -static

No MSYS2 runtime DLLs are packaged with this release. The executables import
Windows system/UCRT API-set DLLs such as KERNEL32.dll and
api-ms-win-crt-*.dll.

The `LICENSES/` directory contains license metadata for the components below:

  LICENSES/DLL_LICENSES.md
  LICENSES/bowtie2/LICENSE
  LICENSES/zlib/LICENSE
  LICENSES/gcc-libs/
